Digital version of my resume on my [site](https://nyashanziramasanga.com/)

## Getting Started

```bash
# Start server
$ cd online-resume
$ npm i
$ npm start
```

## Deployment

```bash
# Run build script for deployment
$ npm run build

# Deploy to netlify if not setup
$ npm install netlify-cli -g
$ netlify deploy
```
