# 📝 Online Resume 📝

Digital version of my resume

## Usage

```bash
# Start server
$ cd online-resume
$ npm i
$ npm start
```

### Deployment

```bash
# Run build script for deployment
$ npm run build

# Deploy to netlify if not setup
$ npm install netlify-cli -g
$ netlify deploy
```

## Tech Used

- [React](https://reactjs.org/)
- [Wakatime Chart](https://wakatime.com/)
